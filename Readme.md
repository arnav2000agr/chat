Alternatively to run this project locally:

Clone this repository.

For Backend:
1. cd backend
2. make a .env file in this directory and add PORT and MONGO_URI
3. npm install
3. npm start

For Frontend:
1. cd frontend
2. npm install
3. npm run dev
4. site will be live on localhost.
